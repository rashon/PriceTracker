# PriceTracker

PriceTracker is an Android application designed to help users track stock prices and view detailed market information in real-time. Built with modern Android development practices, it utilizes Jetpack Compose for the UI and follows a clean architecture pattern.

    *   Price change indicators (Green for positive, Red for negative).
    *   Company name and description.
*   **State Management:** Handles loading states, error scenarios, and successful data retrieval seamlessly.
